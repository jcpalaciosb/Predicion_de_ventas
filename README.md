# Predicion_de_ventas

En el presente proyecto se han aplicado los conocimientos adquiridos sobre los Fundamentos de Data Science, Python, Numpy y Pandas, ejecutando codigos para la revision, evaluacion y tratamiento de la data, asi como graficas que han permitido visualizar el comportamiento de las mismas antes de aplicar los conocimiento de Aprendizaje Automatico Supervisado a traves de dos modelos de regresion: Regresion Lineal y Arbol de Decisiones, evaluando las metricas de cada modelo y seleccionando aquella que tiene los mejores valores en cuanto a coeficiente de determinacion y menores valores de error de los datos resultantes con respecto a su media.

Se incluye una presentacion, que resume lo actuado en este proyecto.
